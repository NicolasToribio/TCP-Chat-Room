Test Instructions - TCP Chat Room with PySimpleGUI
Requires PySimpleGUI package.

Server Side
● Use the terminal to find your IP address to host the session
● Replace the string “host” with your IP address on line 5
● Save the file and run it
● The server is running once the message “server is online” is shown

Client Side
● Run the file
● Type in the IP address of the host you wish to connect to in the first tab, and the
nickname you wish to use in the chat in the second tab
● Press connect to connect to the chat, or exit to close the program.
